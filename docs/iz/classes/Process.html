public class Process : PropertyPublisher

Process encapsulates several useful methods of std.process to make a serializable, synchronous process.

Example

Process prc = construct!Process;
scope(exit) prc.destruct;

prc.executable = "echo";
prc.parameters = "123";
prc.usePipes = true;
prc.execute;
assert(prc.output.readln == "123\n");
